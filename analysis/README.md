# Arctic ROSI Analysis

This folder contains the primary analysis notebooks used for the thesis:

**“Projected Changes in Rain-on-Snow-on-Ice (ROSI) Events Across the Arctic”**

The notebooks in this directory include workflows for historical model evaluation, ROSI event detection, spatial and temporal trend analysis, signal-to-noise calculations, and figure generation.

## Files

### `arctic_project.ipynb`

Primary analysis notebook for the thesis.

This notebook contains the main workflow used to:

- Detect rain-on-snow-on-ice (ROSI) events
- Compare CESM2-LE simulations with observational datasets
- Analyze projected changes in ROSI frequency under SSP3-7.0
- Compute snow-on-ice (SOI) and rain-on-ice (ROI) metrics
- Calculate signal-to-noise ratios (SNR)
- Generate figures and statistics used in the thesis

The notebook integrates processed datasets from:

- NOAA/NSIDC sea ice concentration
- ERA5 precipitation, snow depth, and temperature
- CESM2-LE climate simulations

### `arctic_backup_code.ipynb`

Backup and development notebook containing intermediate analyses, testing workflows, and supporting code used during project development.

This notebook includes exploratory analyses, alternative methods, debugging workflows, and archived code not necessarily used in the final thesis results.

## Notes

Notebook outputs were cleared before upload to reduce file size. Figures and intermediate outputs can be regenerated by running the notebooks with the required processed datasets.

Due to the size of CESM2-LE and ERA5 datasets, raw climate data are not included in this repository.
